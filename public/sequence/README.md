# Frame sequence for NeuralCore X1

The app uses all images in this folder for the scroll animation. Supported pattern:

- `ezgif-frame-001.jpg`, `ezgif-frame-002.jpg`, … (3-digit zero-padded, .jpg)

Scroll position drives which frame is shown (0% scroll = first frame, 100% = last frame). All frames are preloaded before the experience starts.
